# Sideways Tetris

A novel twist on the classic Tetris game - pieces fall **left to right** instead of top to bottom!

## Play Online

**[Play Sideways Tetris](https://rexuvia.com/games/sideways-tetris.html)**

## Features

- 🎮 **Novel Gameplay:** Pieces fall horizontally from left to right
- 🧱 **Complete Vertical Columns:** Clear columns instead of rows
- 🎨 **Futuristic Neon Aesthetic:** Cyan, magenta, and green glowing pieces
- 📱 **Mobile-First Design:** Touch controls optimized for phones
- ⌨️ **Desktop Controls:** Arrow keys + Space to rotate
- 🚀 **Progressive Difficulty:** Speed increases as your score grows
- 💯 **Scoring System:** Bonus points for multi-column clears

## Controls

### Desktop
- **↑/↓ Arrow Keys:** Move piece up/down
- **Space:** Rotate piece
- **→ Arrow Key:** Hard drop (instant drop to right edge)
- **P:** Pause/Resume

### Mobile
- **Swipe Up/Down:** Move piece up/down
- **Tap:** Rotate piece
- **Swipe Right:** Hard drop
- **Pause Button:** Pause/Resume

## How It Differs from Classic Tetris

In traditional Tetris, pieces fall from top to bottom and you complete horizontal rows. In **Sideways Tetris**:

1. Pieces spawn on the **left edge** and fall **toward the right**
2. You move pieces **vertically** (up/down) instead of horizontally
3. Complete **vertical columns** instead of horizontal rows
4. The game ends when pieces stack up to the **left edge**
